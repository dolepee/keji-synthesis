# KEJI x402 Demo Script

## Target Length
- 90 to 120 seconds

## Sequence

1. Open `https://keji-x402.up.railway.app/demo`
   Say:
   "KEJI is an autonomous research CFO. It decides whether a task is worth paying for, buys the context, completes the task, and leaves an onchain receipt."

2. Show the flow on the demo dashboard
   Say:
   "The loop is request, plan, budget check, x402 purchase, Bankr reasoning, verification, and receipt anchoring."

3. Open the report catalog
   Say:
   "KEJI exposes a live research service surface with a public catalog and x402-gated report delivery."

4. Open the agent manifest
   Say:
   "This is the machine-readable agent description used for discoverability and execution context."

5. Open the ERC-8004 Base tx
   Say:
   "KEJI has a real ERC-8004 registration on Base Mainnet."

6. Open one or two Status receipt txs
   Say:
   "Every completed task leaves an anchored receipt on Status Sepolia."

7. Show the repo and logs
   Say:
   "The repo contains the execution log, receipts, treasury state, and the agent manifest."

8. Close with the claim
   Say:
   "KEJI is not just a report bot. It is a budget-aware agent that can justify a paid task, execute it, and produce a verifiable audit trail."
